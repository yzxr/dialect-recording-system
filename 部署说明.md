## 🔧 常见问题

### 1. 注册失败，提示"数据库连接失败"
- 检查 `api/config.php` 中的数据库密码是否正确
- 确认 `luyin_db` 数据库已创建

### 2. 注册失败，提示"表不存在"
- 需要导入 `database.sql` 创建 users 表
- 登录 phpMyAdmin，选择 `luyin_db`，导入 SQL 文件

### 3. 页面空白
- 检查文件权限是否为 755
- 检查 PHP 版本是否 ≥ 7.0
- 查看宝塔面板 → 网站 → PHP 版本

### 4. 跨域错误
- `config.php` 中已配置 CORS，如需修改请编辑该文件
