### 1️⃣ 上传文件到服务器

```bash

# 后端接口

/www/wwwroot/luyin.fzzyd.com/api/config.php

上传 login.php 到：
/www/wwwroot/luyin.fzzyd.com/api/login.php

上传 register.php 到：
/www/wwwroot/luyin.fzzyd.com/api/register.php

上传 upload.php 到：
/www/wwwroot/luyin.fzzyd.com/api/upload.php

上传 export.php 到：
/www/wwwroot/luyin.fzzyd.com/api/export.php

上传 download.php 到：

上传 clean_old_recordings.php 到：
/www/wwwroot/luyin.fzzyd.com/api/clean_old_recordings.php
```

### 2️⃣ 创建 uploads 目录

```bash
# 创建上传目录
mkdir -p /www/wwwroot/luyin.fzzyd.com/uploads

# 设置权限
chmod 755 /www/wwwroot/luyin.fzzyd.com/uploads

# 设置所有者
chown www:www /www/wwwroot/luyin.fzzyd.com/uploads
```

### 3️⃣ 配置数据库

**编辑 `api/config.php`：**

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'luyin_db');      // 数据库名
define('DB_USER', 'luyin_user');    // 数据库用户
define('DB_PASS', 'Luyin2026!');    // 数据库密码
define('DB_CHARSET', 'utf8mb4');
```

### 4️⃣ 导入数据库表结构

**使用 `database.sql` 文件（如有）或手动创建：**

```sql
-- 创建数据库
CREATE DATABASE IF NOT EXISTS `luyin_db` DEFAULT CHARACTER SET utf8mb4;

USE `luyin_db`;

-- 用户表
CREATE TABLE `users` (
    `id` INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    `name` VARCHAR(50) NOT NULL,
    `phone` VARCHAR(20) NOT NULL UNIQUE,
    `company` VARCHAR(100) NOT NULL,
    `employee_id` VARCHAR(50) NOT NULL,
    `email` VARCHAR(100) NOT NULL,
    `password` VARCHAR(255) NOT NULL,
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    `last_login` TIMESTAMP NULL,
    `status` TINYINT DEFAULT 1,
    INDEX `idx_phone` (`phone`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

-- 录音记录表
CREATE TABLE `recordings` (
    `id` INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
    `user_id` INT UNSIGNED NOT NULL,
    `audio_file` VARCHAR(255) NOT NULL,
    `duration` INT DEFAULT 0,
    `status` TINYINT DEFAULT 0,
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (`user_id`) REFERENCES `users`(`id`) ON DELETE CASCADE,
    INDEX `idx_user_id` (`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

### 5️⃣ 配置自动清理任务

**宝塔面板 → 计划任务 → 添加任务**

| 配置项 | 值 |
|--------|-----|
| 任务类型 | Shell 脚本 |
| 执行周期 | 每天 |
| 小时 | 3 |
| 分钟 | 0 |
| 脚本内容 | `php /www/wwwroot/luyin.fzzyd.com/api/clean_old_recordings.php` |

**保存后点击"执行"测试**
