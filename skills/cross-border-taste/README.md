# 🌙 跨境选品决策助手

> "数据是望远镜，但只能看见过去。Taste 是雷达，感知正在发生的变化。"
> 
> **定位：** 选品决策工具 — 帮你提高爆款命中率，减少接盘概率

## 这个 Skill 能帮你什么

如果你做过跨境电商，一定有过这种体验：
- 数据都 OK，但总觉得"哪里不对味"
- 看到一个品，数据还没起来，直觉告诉你"这个能爆"
- 入场后发现，数据好的时候，机会已经没了
- 同样的数据工具，为什么有人命中率就是高？

这个 Skill 基于「Taste 是跨境选品真正的护城河」理念，帮你：

1. **选品决策** — 这个品能不能做？（数据+taste 双重判断）
2. **把模糊的"不对味"说清楚** — 用三问法拆解直觉
3. **识别早期信号** — 在数据滞后前感知趋势
4. **避免接盘** — 判断现在入场是机会还是接盘
5. **训练判断力** — 每周复盘，校准直觉，提高命中率

## 核心理念

### Taste 是什么
- ✅ 对目标用户审美的精准感知
- ✅ 可被论证的判断力
- ✅ 输入量 + 反思的产物

### Taste 不是什么
- ❌ "我觉得这个好看"（个人偏好）
- ❌ 玄学/随机感觉
- ❌ 数据的替代品

### Taste 从哪里来
1. **大量消费目标市场文化内容** — 超前数据 6-12 个月
2. **建立跨类目审美参照系** — 发现传导时间差
3. **训练好坏识别习惯** — 每次选品都追问
4. **与目标用户真实同频** — 在他们活跃的社区泡着

## 使用建议

### 每周 Routine（2-3 小时）— 保持市场敏感度
| 渠道 | 时间 | 追踪内容 |
|------|------|---------|
| Pinterest | 30min | 趋势板、配色、风格 |
| Instagram | 30min | 生活方式博主内容 |
| TikTok | 30min | 爆款视频、亚文化情绪 |
| Reddit | 30min | 垂直社区讨论 |
| 竞品/平台 | 30min | 新品、定价、主图迭代 |

### 每次选品决策流程
1. **先看数据**（销量、竞争、利润、趋势）— 用你的工具
2. **再用 taste 框架**（三问法）— 用这个 skill
3. **记录直觉判断**（对味/不对味/说不清）
4. **做出决策**（做/不做/有条件做）
5. **事后复盘校准**（1-4 周后验证）

### 避免的陷阱
- ❌ 只消费不反思 → 审美停在表面
- ❌ 全数据驱动 → 过滤掉 taste 感知的机会
- ❌ 追求确定答案 → taste 是概率判断，不是预测
- ❌ 只看不记 → 不记录就无法复盘校准

## 常见问题

**Q: Taste 和数据分析冲突时怎么办？**
A: 把冲突写下来，追问"为什么直觉不对"。可能是你发现了数据盲区，也可能是你的 taste 还没校准。记录 + 事后验证，是训练的方式。

**Q: 怎么知道我的 taste 准不准？**
A: 每周复盘，统计判断正确率。3 个月后，你会看到明显的校准曲线。

**Q: 新手能培养 taste 吗？**
A: 能。Taste 是输入量的产物，不是天赋。关键是：大量消费 + 刻意反思。
