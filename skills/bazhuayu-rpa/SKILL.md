# 八爪鱼 RPA 集成技能

## 功能描述
集成八爪鱼 RPA 平台，让 OpenClaw 能够调用 RPA 机器人执行自动化任务。

## 触发条件
当用户提到以下关键词时激活：
- "八爪鱼 RPA"
- "RPA 自动化"
- "运行 RPA"
- "执行流程"
- "bazhuayu"

## 核心能力

### 1. RPA 任务执行
- 调用八爪鱼 API 执行指定流程
- 监控任务执行状态
- 获取执行结果

### 2. 流程管理
- 列出可用 RPA 流程
- 查看流程详情
- 启停流程

### 3. 数据对接
- 接收 RPA 执行结果
- 转发数据到其他系统
- 生成执行报告

## 配置要求

### 环境变量
```bash
BAZHUAYU_API_KEY=你的 API 密钥
BAZHUAYU_ACCOUNT_ID=账户 ID
```

### 依赖工具
- curl (API 调用)
- jq (JSON 处理)

## 使用示例

```
用户：用八爪鱼 RPA 执行客户数据整理流程
助手：好的，正在调用八爪鱼 RPA 执行"客户数据整理"流程...
```

## 注意事项
- 需要先完成八爪鱼账号注册和试用申请
- API 密钥需要妥善保管
- 部分高级功能需要企业版授权

## 相关文件
- `scripts/rpa-execute.sh` - RPA 任务执行脚本
- `scripts/rpa-status.sh` - 任务状态查询脚本
- `scripts/rpa-list.sh` - 流程列表查询脚本
