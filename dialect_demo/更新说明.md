# 方言采集系统 - 员工导入格式更新

## 📋 更新内容

### 新的员工表格字段格式：

| 字段 | 必填/可选 | 说明 |
|------|----------|------|
| 发音编号 | 可选 | 员工编号/工号 |
| 姓名 | ✅ 必填 | 员工真实姓名 |
| 手机号 | ✅ 必填 | 登录账号 |
| 密码 | ✅ 必填 | 登录密码 |
| 年龄 | 可选 | 默认 30 岁 |
| 方言能力 | 可选 | 如"闽南语，粤语" |
| 备注 | 可选 | 其他说明信息 |

---

## 🚀 部署步骤

### 1️⃣ 上传文件到服务器

在宝塔面板中，进入 `/www/wwwroot/fy.fzzyd.com/` 目录，上传并覆盖以下文件：

- `database.py`
- `app.py`
- `templates/import_users.html`

### 2️⃣ 重启 Python 项目

在宝塔 **Python 项目管理器** 中点击 **"重启"**

### 3️⃣ 验证

访问 http://fy.fzzyd.com/users/import

查看页面底部提示是否更新为：
```
Excel 格式（表头）：
发音编号 | 姓名 | 手机号 | 密码 | 年龄 | 方言能力 | 备注

必填字段：姓名、手机号、密码
可选字段：发音编号、年龄、方言能力、备注
```
