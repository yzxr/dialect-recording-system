## 📋 使用流程

### 1️⃣ 登录系统
用管理员账号登录

### 2️⃣ 导入员工
- 点击"导入员工"
- 使用 Excel 模板或 CSV 文件
- 格式：姓名，手机号，密码，邮箱，年龄，方言能力

**测试数据（CSV 格式）：**
```csv
姓名，手机号，密码，邮箱，年龄，方言能力
张三，13800138001,123456,zhang@example.com,35，"闽南语，粤语"
李四，13800138002,123456,li@example.com,28，客家话
王五，13800138003,123456,wang@example.com,45，闽南语
```

### 3️⃣ 导入话术
- 点击"导入话术"
- 选择方言类型（闽南语/客家话/粤语）
- 粘贴话术内容（每行一条）
- 系统自动分配给员工

**测试话术：**
```
你好，欢迎来咨询
我们这个效果很好的
很多客户都反馈不错
今天有优惠活动
您想了解哪方面呢
```

### 4️⃣ 员工登录
- 用导入的手机号和密码登录
- 点击"我的任务"
- 点击"上传录音"
- 上传音频文件（文件名格式：方言类型_姓名_话术编号_时间戳.wav）

**文件名示例：**
```
闽南语_张三_S001_202604071030.wav
粤语_李四_S002_202604071145.wav
```

### 5️⃣ 审核录音
- 管理员点击"审核管理"
- 查看待审核列表
- 检查文件名是否规范
- 通过或驳回

### 6️⃣ 批量下载
- 管理员点击"批量下载"
- 打包下载所有已通过的录音
- 下载后可点击"删除已下载文件"释放空间

## ⚠️ 注意事项

1. **Demo 版限制：**
   - 单团队模式（不支持多团队）
   - 手动回收（无自动定时任务）
   - 无微信通知（仅本地操作）
   - 无积分系统

2. **录音文件存储：**
   - 本地存储：`static/uploads/`
   - 记得定期下载并删除

3. **数据库：**
   - SQLite 单文件数据库
   - 位置：`instance/dialect_demo.db`
   - 可删除重置
