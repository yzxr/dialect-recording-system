## 一、微信公众号

### 账号类型选择

| 类型 | 推送频率 | 优势 | 劣势 | 建议 |
|-----|---------|------|------|------|
| **服务号** | 每月 4 次 | 支持更多接口（预约、支付、模板消息） | 推送次数少 | ✅ 推荐（适合转化） |
| **订阅号** | 每天 1 次 | 内容曝光高 | 功能受限，折叠在订阅号文件夹 | 备选（适合纯内容引流） |

### 注册材料准备

- [ ] 企业营业执照（或个体工商户）
- [ ] 法人身份证
- [ ] 对公账户（用于认证）
- [ ] 运营者身份证 + 手机号
- [ ] 公众号名称（建议 2-3 个备选）
- [ ] 公众号头像（300x300px）
- [ ] 认证费用：300 元/年

### 命名建议

**方向一：品牌 + 品类**
- 雅杰扁针抗衰
- 雅杰中医美容

**方向二：专家 IP + 品类**
- XX 老师扁针抗衰
- XX 中医抗衰研习社

**方向三：效果导向**
- 原生年轻抗衰
- 自然焕颜实验室

### 注册流程

```
1. 登录 https://mp.weixin.qq.com
2. 点击"立即注册" → 选择账号类型
3. 填写邮箱、密码 → 激活
4. 信息登记（主体信息、运营者信息）
5. 填写公众号信息（名称、介绍、头像）
6. 微信认证（需 300 元费用，1-3 个工作日）
```

### 认证后功能

- ✅ 自定义菜单
- ✅ 自动回复
- ✅ 用户管理
- ✅ 模板消息（服务号）
- ✅ 微信支付（服务号）
- ✅ 预约小程序关联

## 二、小红书账号

### 账号类型

| 类型 | 适合场景 | 认证要求 |
|-----|---------|---------|
| **个人号** | 初期试水、IP 打造 | 无门槛 |
| **专业号** | 品牌官方、商业推广 | 营业执照 + 行业资质 |
| **企业号** | 多账号矩阵、员工号 | 营业执照 |

**建议**：先用个人号/专业号起步，后期再考虑企业号矩阵

### 注册材料准备

- [ ] 手机号（未注册过小红书）
- [ ] 账号名称（2-3 个备选）
- [ ] 头像（建议真人/品牌 logo）
- [ ] 简介（50 字内，突出价值）
- [ ] 专业号认证：营业执照 + 行业资质（可选）

### 命名建议

**小红书风格更偏向：**
- 亲切感 + 专业感结合
- 可加入 emoji 增加辨识度

**示例：**
- 🌿 雅杰扁针抗衰
- 月仙·中医抗衰笔记
- 扁针抗衰｜雅杰
- 中医抗衰 XX 老师

### 简介模板

```
📍 专注中医扁针抗衰 X 年
✨ 无手术·无填充·自然焕颜
🏥 已帮助 XXX+ 客户见证原生年轻
📩 面诊/合作请私信
```

### 注册流程

```
1. 下载小红书 APP
2. 手机号注册/登录
3. 完善个人资料（头像、昵称、简介）
4. 专业号认证（可选，路径：我 → 设置 → 专业号中心）
```

## 三、账号运营建议

### 公众号（深度内容 + 转化）

| 内容类型 | 频率 | 目的 |
|---------|------|------|
| 科普长文 | 每周 1-2 篇 | 建立专业信任 |
| 案例展示 | 每周 1 篇 | 效果证明 |
| 活动通知 | 按需 | 转化引导 |

### 小红书（种草 + 引流）

| 内容类型 | 频率 | 目的 |
|---------|------|------|
| 效果对比图 | 每周 2-3 篇 | 视觉冲击引流 |
| 科普短视频 | 每周 1-2 篇 | 建立专业形象 |
| 客户见证 | 每周 1 篇 | 信任背书 |

### 引流路径设计

```
小红书笔记 → 主页留微信号/公众号 → 私域沉淀 → 1v1 咨询 → 到店体验
公众号文章 → 文末二维码 → 加企微领资料 → 私域运营 → 转化
```

## 四、注意事项

### 医疗相关合规

- ⚠️ 避免绝对化用语（"根治""100% 有效""最"等）
- ⚠️ 案例展示需客户授权
- ⚠️ 小红书医疗内容审核较严，注意措辞
- ⚠️ 公众号医疗类内容需相关资质认证

### 建议话术调整

| 原话术 | 调整后 |
|-------|-------|
| "根治皱纹" | "改善皱纹" |
| "100% 有效" | "多数客户反馈有效" |
| "最好/第一" | "专注/深耕 X 年" |
| "治疗" | "调理/养护" |
