# 📦 录音平台新增功能部署说明

## 🆕 新增功能

### 1️⃣ 话术栏模块
- ✅ 显示需要照着念的文本
- ✅ 可滚动阅读
- ✅ 内容可在 HTML 中修改

### 2️⃣ 录音存储功能
- ✅ 录音文件保存为 WAV 格式
- ✅ 存储到服务器 `uploads/` 目录
- ✅ 按日期分类存储（`uploads/2026-04-05/`）
- ✅ 自动关联到用户账号
- ✅ 数据库记录录音信息

---

## 📋 部署步骤

### 1️⃣ 上传文件

**上传以下文件到服务器：**

```bash
# 1. 更新录音页面
上传 record.html 到：
/www/wwwroot/luyin.fzzyd.com/frontend/record.html

# 2. 上传后端接口
上传 upload.php 到：
/www/wwwroot/luyin.fzzyd.com/api/upload.php
```

---

### 2️⃣ 创建上传目录

**SSH 执行：**
```bash
# 创建 uploads 目录
mkdir -p /www/wwwroot/luyin.fzzyd.com/uploads

# 设置权限
chmod 755 /www/wwwroot/luyin.fzzyd.com/uploads

# 设置所有者（如果需要）
chown www:www /www/wwwroot/luyin.fzzyd.com/uploads
```

---

### 3️⃣ 修改 PHP 配置（如果需要）

**编辑 `/www/server/php/74/etc/php.ini`：**

```ini
; 确保以下配置正确
upload_max_filesize = 50M
post_max_size = 50M
max_execution_time = 300
```

**重启 PHP：**
```bash
/etc/init.d/php74-fpm restart
```

---

### 4️⃣ 测试功能

**1. 访问录音页面：**
```
http://luyin.fzzyd.com/frontend/record.html
```

**2. 登录后查看：**
- 左侧应该显示话术栏
- 右侧显示录音按钮

**3. 点击"开始录音"：**
- 允许浏览器使用麦克风
- 开始朗读话术栏内容
- 点击"停止录音"

**4. 检查上传结果：**
- 显示"录音已保存"
- 显示文件名

**5. 检查服务器文件：**
```bash
ls -la /www/wwwroot/luyin.fzzyd.com/uploads/
```

### 录音文件存储

**存储路径：**
```
/www/wwwroot/luyin.fzzyd.com/uploads/2026-04-05/recording_13800138000_1712318400_abc123.wav
```

**文件名格式：**
```
recording_手机号_时间戳_随机字符串.wav
```

**数据库记录：**
- 表：`recordings`
- 字段：`user_id`, `audio_file`, `duration`, `status`, `created_at`
